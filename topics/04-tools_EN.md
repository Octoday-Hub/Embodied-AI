## 🛠️ Tools & Platforms

> A curated toolkit for robot development, covering simulation environments, development frameworks, SLAM stacks, and vision tools.
>
> This list is continuously updated. If you find a useful tool that is missing, or if an existing entry needs revision, please submit a [PR](https://github.com/Octoday-Hub/Embodied-AI/pulls) or open an [Issue](https://github.com/Octoday-Hub/Embodied-AI/issues).
