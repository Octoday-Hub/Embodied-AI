# 贡献指南

感谢你愿意为 Octoday Hub 贡献力量！我们致力于打造一份高质量的具身智能领域资源索引，你的每一次贡献都能帮助更多开发者。

## 如何贡献

你可以通过以下方式参与：

- **直接提交 PR**：修改或新增内容，按照 PR 模板提交。
- **提交 Issue**：报告错误、建议添加新资源、提出改进意见。

## 贡献流程

### 1. 准备环境

- 确保你有一个 GitHub 账号。
- Fork 本仓库到你的账号下。
- 克隆你的 Fork 到本地（如果需要批量编辑，也可直接在网页端修改）。

### 2. 创建分支

从 main 分支切出一个新分支，命名规则：

- `update-basics-[主题]` 对应 `00-basics.md`
- `add-company-[公司名]` 对应 `01-companies.md`
- `update-jobs-[公司名]` 对应 `02-jobs.md`
- `update-paper-[论文名]` 对应 `03-papers.md`
- `update-tools-[工具名]` 对应 `04-tools.md`

### 3. 修改内容

请遵循以下格式规范：

#### 认知基座 (00-basics.md)

- 推荐书籍需提供**书名**、**作者**、**出版年份**、**推荐理由**（可选）。
- 推荐课程需提供**课程名称**、**机构**、**链接**、**简介**。

#### 公司库 (01-companies.md)

- 提供**公司名称**、**官网链接**、**一句话简介**、**技术方向**、**地点**。
- 保持客观中立，一句话概括核心业务。

#### 招聘信息 (02-jobs.md)

- 提供**岗位名称**、**公司**、**地点**、**类型**（全职/实习/校招）、**投递链接**。
- 招聘信息需标注**截止日期**或**更新时间**，过期信息请及时清理。
- 使用统一的表格格式，保持排版对齐。

#### 论文合集 (03-papers.md)

- 提供**论文标题**、**作者**、**发表会议/期刊**、**年份**、**代码链接**（如有）。
- 尽量附上**一句话总结**，说明核心贡献。
- 按研究方向分类（如 Manipulation、Locomotion、Navigation 等）。

#### 工具矩阵 (04-tools.md)

- 提供**工具名称**、**类型**（仿真/控制/数据集/框架）、**官方链接**、**一句话简介**。
- 标注是否开源及主要编程语言。

### 4. 提交 PR

- 确保修改内容准确、有据可查。
- 一个 PR 尽量只针对一个明确的修改。
- 填写 PR 模板中的检查清单。

## 信息准确性

- **可验证**：提供的信息应能在官网或权威渠道核实。
- **时效性**：招聘、投融资等信息请尽量提供最新内容。
- **无偏见**：保持客观中立。

## 行为准则

- 保持友善与尊重。
- 尊重知识产权。
- 商业合作请通过邮件联系，避免在 Issue/PR 中发布广告。
