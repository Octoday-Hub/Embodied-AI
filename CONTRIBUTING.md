# 贡献指南

感谢你愿意为 **Octoday-Embodied-Hub** 贡献力量！我们致力于打造一份高质量的具身智能领域资源索引，你的每一次贡献都能帮助更多开发者。

## 如何贡献

你可以通过以下方式参与：

- **直接提交 PR**：修改或新增内容，按照 PR 模板提交。
- **提交 Issue**：报告错误、建议添加新资源、提出改进意见。

## 贡献流程

### 1. 准备环境
- 确保你有一个 GitHub 账号。
- Fork 本仓库到你的账号下。
- 克隆你的 Fork 到本地（如果需要批量编辑，也可直接在网页端修改）。

### 2. 创建分支
从 `main` 分支切出一个新分支，命名规则：
- `add-company-[公司名]`
- `update-paper-[论文名]`
- `fix-typo-[文件名]`
- 等

### 3. 修改内容
请遵循以下格式规范：

#### 公司库 (`01-companies.md`)
每个公司条目格式：
```markdown
### [公司名称]
- **官网**：[链接]
- **简介**：一句话介绍
- **技术方向**：人形机器人 / 具身大模型 / 核心部件 / ...
- **地点**：城市，国家
